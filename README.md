# HFPAC
HFPAC stands for High-Fidelity Python Audio Codec, and as the name suggests, is a codec that prioritises quality whilst providing as much compression as possible.
This repository includes the HFPAC format itself, as well as the actual codec, related algorithms used in the codec, the HFPAC Encoder & its GUI, and the HFPAC Player & its GUI - as well as HFPAC's main entry point (main.py).

The first two version numbers of the Encoder and Player are directly linked to the HFPAC format version, e.g. Player 6.1.x.y will play up to v6.1 files, and Encoder 6.0.x.y will encode up to v6 files.
